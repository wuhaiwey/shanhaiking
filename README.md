# 🎮 山海经王者荣耀 - Unity 6 项目

## 📁 项目结构

```
Assets/
├── _Project/                    # 项目核心文件
│   ├── Scripts/                 # 脚本
│   │   ├── Core/               # 核心系统
│   │   ├── Hero/               # 英雄系统
│   │   ├── Skill/              # 技能系统
│   │   ├── Battle/             # 战斗系统
│   │   ├── UI/                 # UI系统
│   │   └── Network/            # 网络系统
│   ├── Prefabs/                # 预制体
│   ├── Scenes/                 # 场景
│   ├── Resources/              # 资源
│   └── Art/                    # 美术资源
├── Plugins/                     # 插件
└── Settings/                    # 设置
```

## 🚀 核心系统

### 1. 英雄系统 (HeroSystem)
- 属性管理（生命、法力、攻击、防御）
- 等级成长
- 装备系统

### 2. 技能系统 (SkillSystem)
- 主动技能
- 被动技能
- 技能冷却
- 技能特效

### 3. 战斗系统 (BattleSystem)
- 伤害计算
- 命中判定
- 暴击系统
- 控制效果

### 4. 移动系统 (MovementSystem)
- 摇杆控制
- 自动寻路
- 闪避/冲刺

### 5. 网络系统 (NetworkSystem)
- Photon PUN2 联机
- 状态同步
- 延迟补偿

## 🐉 英雄设计

### 后羿 (射手)
```csharp
// 基础属性
生命值: 3200 (+180/级)
法力值: 500 (+30/级)
攻击力: 180 (+12/级)
防御力: 80 (+5/级)
移动速度: 360

// 技能
Q: 连珠箭 - 快速射出3支箭
W: 穿云箭 - 穿透伤害
E: 逐日步 - 翻滚位移
R: 射日 - 全图大招
```

### 九尾狐 (刺客)
```csharp
// 基础属性
生命值: 2800 (+150/级)
法力值: 600 (+40/级)
攻击力: 200 (+15/级)
防御力: 60 (+4/级)
移动速度: 380

// 技能
Q: 狐火 - 范围伤害
W: 魅惑 - 控制技能

R: 九尾分身 - 召唤分身
```
