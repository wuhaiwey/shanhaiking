## 🚀 使用指南

### 打开项目
```
1. 打开 Unity Hub
2. 选择 ShanHaiKing 项目
3. 打开 GameScene 场景
4. 点击 Play 运行
```

### 操作说明
- **WASD**: 移动
- **QWER**: 技能
- **鼠标**: 选择目标
- **滚轮**: 缩放视角
