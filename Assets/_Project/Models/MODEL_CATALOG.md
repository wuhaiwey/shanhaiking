## 💡 查看方法

### 方法1: Blender
1. 打开 Blender
2. 文件 → 导入 → FBX
3. 选择对应的 .fbx 文件

### 方法2: Unity
1. 将FBX拖入Unity项目
2. 在Scene中查看

### 方法3: 渲染脚本
运行 `render_nine_views.py` 生成九视图PNG:
```bash
blender --background --python render_nine_views.py
```
