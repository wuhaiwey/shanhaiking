# 🎮 山海经王者荣耀 - Unity 项目启动指南

## 📥 Unity 安装步骤

### 1. Unity Hub 已安装 ✅
Unity Hub 应用程序已安装到 `/Applications/Unity Hub.app`

### 2. 登录 Unity 账号
**请打开 Unity Hub 并登录您的 Unity 账号。**

如果没有账号：
1. 访问 https://unity.com/
2. 点击 "Get Started"
3. 使用邮箱注册

### 3. 安装 Unity 2022.3 LTS

**在 Unity Hub 中：**
1. 点击左侧 "Installs"
2. 点击右上角的 "Install Editor"
3. 选择 "Unity 2022.3 LTS"
4. 勾选以下组件：
   - ✅ Mac Build Support (Mono)
   - ✅ iOS Build Support
   - ✅ Android Build Support
   - ✅ Visual Studio Code Editor
5. 点击 "Install"

⏳ **下载大约需要 3-5 GB 流量，15-30 分钟**
