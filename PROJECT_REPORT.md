## 💡 如何继续开发

### 1. 打开项目
```
1. 打开 Unity Hub
2. 点击 "Open"
3. 选择 /Users/mili/Desktop/ShanHaiKing
4. 等待加载完成
```

### 2. 创建测试场景
```
1. File -> New Scene
2. 保存为 "TestScene"
3. 添加一个 Plane 作为地面
4. 添加 Directional Light
5. 创建一个空物体命名为 "GameManager"
6. 给 GameManager 添加 GameManager.cs 脚本
7. 创建一个 Capsule 命名为 "HouYi"
8. 给 HouYi 添加 Hero_HouYi.cs 脚本
9. 把摄像机拖到 GameCamera 脚本
10. 点击 Play 运行
```
